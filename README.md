# MetodosNumericos_Jul2026

Repositorio para el curso de invierno de Metodos Numericos en UMSS, Jul - 2026

## instalacion

Usar conda + mamba env para instalar las librerias y reproducir el codigo

1. instalar conda
1. en el base environment de conda instalar mamba `conda install mamba -n base -c conda-forge`
1. crear el environment para este rep `mamba env create -f environment.yml`
1. cuando necesitemos actualizar el environment usar `mamba env update -f environment.yml --prune`
1. para activar el environment usar `conda activate numericalMethods_2026`
